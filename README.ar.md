# 📁 Folder Organizer (منظّم المجلدات)

**أداة محلية وخاصة ومدعومة بالذكاء الاصطناعي تُصنّف أكثر مجلداتك فوضى بأمان — دون أن تكسر تطبيقاتك أو مشاريعك البرمجية أو قواعد بياناتك — مع إمكانية التراجع الكامل بنقرة واحدة.**

كل شيء يعمل على جهازك. لا يُرفع أي شيء إلى الإنترنت. ولا يُنقل أي ملف حتى تُراجع الخطة وتؤكّدها.

## ✨ ما الذي يميّزه

معظم "منظّمات الملفات" تُفرّغ كل شيء في مجلدات حسب الامتداد فقط، فتُعطّل الأشياء التي لا تعمل إلا ككل: مشروع برمجي، تطبيق محمول، قاعدة بيانات، صفحة ويب محفوظة. هذا التطبيق يفهم البنية:

- 🛡️ **يُبقي الوحدات كاملةً سليمة.** المشاريع البرمجية (`package.json`، `.git`، …)، والتطبيقات والبرامج (التطبيقات المحمولة والمتصفحات والمحاكيات والألعاب — حتى ملفات لينكس/ماك التنفيذية بلا امتداد)، وقواعد البيانات (SQLite، LevelDB/RocksDB)، وصفحات الويب المحفوظة (مجلدات `..._files`) تُنقل **كقطعة واحدة** ولا تُبعثر أبدًا.
- 🤖 **مدعوم بالذكاء الاصطناعي، وليس قائمة ثابتة.** الفئات ليست مُبرمجة مسبقًا. يقوم **نموذج لغوي محلي (اختياري)** بتصنيف الأنواع غير المعروفة مع سياق المسار الكامل ويمكنه ابتكار فئات جديدة — وفي نهاية كل فحص **يُعيد مراجعة الخطة بالكامل لتصحيح أخطائه**.
- ⏪ **قابل للتراجع بالكامل.** تُلتقط لقطة لكل عملية؛ ونقرة واحدة تُعيد كل شيء إلى مكانه بالضبط.
- 💾 **مقاوم للأعطال.** إذا انطفأ الحاسوب أو التطبيق في منتصف العملية، يُكمل من حيث توقّف تمامًا.
- 🔒 **محلي وخاص 100٪.** بلا سحابة، ولا إرسال بيانات، ولا حساب.

## 🚀 البدء السريع

```bash
npm install
npm start
```

ثم افتح **http://localhost:4173** في متصفحك.
(لتغيير المنفذ: `PORT=5000 npm start`)

## 🧩 المزايا

- 🗂️ **تصنيف ذكي** — صور، فيديو، موسيقى، مستندات، شيفرة، **نماذج** (أوزان تعلّم آلي)، **قواعد بيانات**، أرشيفات، **اختصارات**، **إعدادات**، والمزيد — بالإضافة إلى أي فئة تُنشئها أنت أو الذكاء الاصطناعي.
- 🛡️ **الحفاظ على تكامل** المشاريع والتطبيقات وقواعد البيانات وصفحات الويب المحفوظة.
- 🤖 **نموذج لغوي محلي (اختياري)** — تصنيف الأنواع غير المعروفة و**مراجعة نهائية لكل الملفات**.
- 🔁 **كشف التكرارات** — الدقيقة (عبر بصمة المحتوى) والصور المتشابهة بصريًا.
- 🧹 **تنظيف** `node_modules` وذاكرات البناء المؤقتة وملفات نظام التشغيل غير المرغوبة.
- 📸 **مجلدات فرعية ذكية** — الصور حسب تاريخ EXIF، والموسيقى حسب الفنان/الألبوم.
- 👀 **مراجعة قبل النقل** — شاهد الخطة كاملةً، وحرّرها، واستثنِ ما تشاء.
- ⏪ **تراجع + لقطات** لبنية المجلدات لكل عملية.
- 💾 **استئناف تلقائي** بعد الانقطاع.
- 📊 **تقدّم مباشر وسجلّ لحظي.**

## 🔒 الخصوصية

هذه أداة شخصية لسطح المكتب: بلا تسجيل دخول، ولا تعدّد مستخدمين، ولا سحابة، ولا تتبّع. تُخزَّن كل البيانات (قواعد التصنيف المكتسبة، اللقطات، سجل العمليات) في مجلد محلي `data/` بجوار التطبيق. وإذا فعّلت النموذج اللغوي الاختياري، تذهب الطلبات فقط إلى النقطة المحلية التي **تُعدّها أنت**.
