# 📁 Folder Organizer (سازمان‌دهندهٔ پوشه)

**ابزاری محلی، خصوصی و مبتنی بر هوش مصنوعی که شلوغ‌ترین پوشه‌های شما را با اطمینان دسته‌بندی می‌کند — بدون آنکه برنامه‌ها، پروژه‌های کد یا پایگاه‌های دادهٔ شما را خراب کند — و با قابلیت بازگردانی کامل با یک کلیک.**

همه‌چیز روی دستگاه خودتان اجرا می‌شود. هیچ چیزی آپلود نمی‌شود. تا زمانی که بازبینی و تأیید نکنید، هیچ فایلی جابه‌جا نمی‌شود.

## ✨ چه چیزی آن را متفاوت می‌کند

بیشتر «سازمان‌دهنده‌های فایل» همه‌چیز را فقط بر اساس پسوند در پوشه‌ها می‌ریزند و در این کار چیزهایی را که فقط به‌صورت یکپارچه کار می‌کنند **خراب می‌کنند**: یک پروژهٔ کد، یک برنامهٔ قابل‌حمل، یک پایگاه داده، یک صفحهٔ وب ذخیره‌شده. این برنامه ساختار را می‌فهمد:

- 🛡️ **واحدهای کامل را دست‌نخورده نگه می‌دارد.** پروژه‌های کد (`package.json`، `.git`، …)، برنامه‌ها و اپلیکیشن‌ها (برنامه‌های قابل‌حمل، مرورگرها، شبیه‌سازها، بازی‌ها — حتی فایل‌های اجرایی لینوکس/مک بدون پسوند)، پایگاه‌های داده (SQLite، LevelDB/RocksDB) و صفحات وب ذخیره‌شده (پوشه‌های `..._files`) به‌صورت **یک‌تکه** جابه‌جا می‌شوند و هرگز پخش نمی‌شوند.
- 🤖 **مبتنی بر هوش مصنوعی، نه یک فهرست ثابت.** دسته‌ها از پیش تعیین‌شده نیستند. یک **LLM محلی (اختیاری)** انواع ناشناختهٔ فایل را با در نظر گرفتن مسیر کامل دسته‌بندی می‌کند و می‌تواند دسته‌های جدید بسازد — و در پایان هر اسکن، **کل نقشه را دوباره بازبینی می‌کند تا اشتباهات خودش را اصلاح کند**.
- ⏪ **کاملاً برگشت‌پذیر.** از هر اجرا یک نسخهٔ فوری گرفته می‌شود؛ با یک کلیک همه‌چیز دقیقاً به جای اولش برمی‌گردد.
- 💾 **مقاوم در برابر قطعی.** اگر رایانه یا برنامه در میانهٔ کار خاموش شود، دقیقاً از همان‌جا که مانده بود ادامه می‌دهد.
- 🔒 **۱۰۰٪ محلی و خصوصی.** بدون ابر، بدون ارسال داده، بدون حساب کاربری.

## 🚀 شروع سریع

```bash
npm install
npm start
```

سپس در مرورگر **http://localhost:4173** را باز کنید.
(برای تغییر پورت: `PORT=5000 npm start`)

## 🧩 امکانات

- 🗂️ **دسته‌بندی هوشمند** — تصاویر، ویدیو، موسیقی، اسناد، کد، **مدل‌ها** (وزن‌های یادگیری ماشین)، **پایگاه‌های داده**، آرشیوها، **میان‌برها**، **تنظیمات** و بیشتر — به‌علاوهٔ هر دسته‌ای که شما یا هوش مصنوعی بسازید.
- 🛡️ **حفظ یکپارچگی** پروژه‌ها، برنامه‌ها، پایگاه‌های داده و صفحات وب ذخیره‌شده.
- 🤖 **LLM محلی (اختیاری)** — دسته‌بندی انواع ناشناخته و **یک بازبینی نهایی روی همهٔ فایل‌ها**.
- 🔁 **تشخیص موارد تکراری** — دقیق (بر اساس هش محتوا) و عکس‌های از نظر بصری مشابه.
- 🧹 **پاک‌سازی** `node_modules`، کش‌های ساخت و فایل‌های زائد سیستم‌عامل.
- 📸 **زیرپوشه‌های هوشمند** — عکس‌ها بر اساس تاریخ EXIF، موسیقی بر اساس هنرمند/آلبوم.
- 👀 **بازبینی پیش از جابه‌جایی** — کل نقشه را ببینید، ویرایش کنید، مستثنا کنید.
- ⏪ **بازگردانی + نسخه‌های فوری** از ساختار پوشه‌ها برای هر اجرا.
- 💾 **ازسرگیری پس از قطعی** به‌صورت خودکار.
- 📊 **پیشرفت زنده + گزارش لحظه‌ای.**

## 🔒 حریم خصوصی

این یک ابزار شخصی رومیزی است: بدون ورود، بدون چند کاربره، بدون ابر، بدون تله‌متری. همهٔ داده‌ها (قوانین دسته‌بندی آموخته‌شده، نسخه‌های فوری، تاریخچهٔ اجرا) در یک پوشهٔ محلی `data/` کنار برنامه ذخیره می‌شوند. اگر LLM اختیاری را فعال کنید، درخواست‌ها فقط به نقطهٔ پایانی محلی‌ای می‌روند که **خودتان** تنظیم کرده‌اید.
