# 📁 Folder Organizer（フォルダー整理ツール）

**ローカルで動作する、プライベートかつAI支援のツール。散らかったフォルダーを安全にカテゴリー分けします — アプリ、コードプロジェクト、データベースを決して壊さず — ワンクリックで完全に元へ戻せます。**

すべてはあなたのマシン上で動作します。何もアップロードされません。あなたが確認して承認するまで、ファイルは一切動きません。

## ✨ 何が違うのか

多くの「ファイル整理ツール」は拡張子だけでフォルダーに振り分けるため、全体でしか機能しないもの — コードプロジェクト、ポータブルアプリ、データベース、保存済みWebページ — を**壊して**しまいます。このアプリは構造を理解します：

- 🛡️ **一体のユニットをそのまま保持。** コードプロジェクト（`package.json`、`.git` など）、アプリケーションやプログラム（ポータブルアプリ、ブラウザー、エミュレーター、ゲーム — 拡張子のないLinux/macOSバイナリーも）、データベース（SQLite、LevelDB/RocksDB）、保存済みWebページ（`..._files` フォルダー）は**一つのまとまり**として移動し、決してバラバラにしません。
- 🤖 **固定リストではなくAI駆動。** カテゴリーは固定ではありません。オプションの**ローカルLLM**が、フルパスの文脈を踏まえて未知のファイル種別を分類し、新しいカテゴリーも作成できます — さらにスキャンの最後に**計画全体を再確認して自らの誤りを修正**します。
- ⏪ **完全に元に戻せる。** 実行ごとにスナップショットを保存。ワンクリックですべて元の場所へ正確に戻します。
- 💾 **クラッシュに強い。** PCやアプリが途中で落ちても、再起動時に中断した箇所から正確に再開します。
- 🔒 **100%ローカルでプライベート。** クラウドなし、テレメトリーなし、アカウント不要。

## 🧩 できること

- 🗂️ **スマートな分類** — 画像、動画、音楽、ドキュメント、コード、**モデル**（ML重み）、**データベース**、アーカイブ、**ショートカット**、**設定** など — あなたやAIが作る任意のカテゴリーも。
- 🛡️ **そのまま保持**：プロジェクト、アプリ、データベース、保存済みWebページ。
- 🤖 **ローカルLLM（オプション）** — 未知の種別を分類し、**全ファイルの最終レビュー**を実施。
- 🔁 **重複検出** — 完全一致（内容ハッシュ）と視覚的に似た写真。
- 🧹 **クリーンアップ** — `node_modules`、ビルドキャッシュ、OSのゴミファイル。
- 📸 **スマートなサブフォルダー** — 写真はEXIF日付、音楽はアーティスト/アルバム。
- 👀 **移動前にレビュー** — 計画全体を確認・編集・除外。
- ⏪ **ロールバック＋スナップショット**（実行ごとのフォルダー構造）。
- 💾 **中断後に自動再開。**
- 📊 **リアルタイムの進捗とログ。**

## 🔒 プライバシー

これは個人向けデスクトップユーティリティです：ログインなし、マルチユーザーなし、クラウドなし、テレメトリーなし。すべての状態（学習した分類ルール、スキャンのスナップショット、取り消し用の実行履歴）はアプリの隣のローカル `data/` フォルダーに保存されます。オプションのLLMを有効にした場合、リクエストは**あなたが**設定したローカルのエンドポイントにのみ送られます。
