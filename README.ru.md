# 📁 Folder Organizer (Организатор папок)

**Локальный, приватный инструмент с поддержкой ИИ, который безопасно раскладывает самые захламлённые папки по категориям — не ломая ваши приложения, кодовые проекты и базы данных — с полным откатом в один клик.**

Всё работает на вашем компьютере. Ничего не загружается в облако. Ни один файл не перемещается, пока вы не просмотрите и не подтвердите план.

## ✨ Чем он отличается

Большинство «органайзеров файлов» просто раскидывают всё по папкам на основе расширения и при этом **ломают** то, что работает только целиком: кодовый проект, портативное приложение, базу данных, сохранённую веб-страницу. Это приложение понимает структуру:

- 🛡️ **Сохраняет целостные единицы.** Кодовые проекты (`package.json`, `.git`, …), приложения и программы (портативные приложения, браузеры, эмуляторы, игры — даже исполняемые файлы Linux/macOS без расширения), базы данных (SQLite, LevelDB/RocksDB) и сохранённые веб-страницы (папки `..._files`) перемещаются **как единое целое** и никогда не разбиваются.
- 🤖 **На базе ИИ, а не фиксированного списка.** Категории не зашиты в коде. Опциональная **локальная LLM** классифицирует неизвестные типы с учётом полного пути и может создавать новые категории — а в конце каждого сканирования **заново проверяет весь план, исправляя собственные ошибки**.
- ⏪ **Полностью обратимо.** Для каждого запуска создаётся снимок; один клик возвращает всё точно на свои места.
- 💾 **Устойчивость к сбоям.** Если ПК или приложение выключится посреди процесса, после перезапуска работа продолжится ровно с того же места.
- 🔒 **100% локально и приватно.** Без облака, без телеметрии, без учётной записи.

## 🚀 Быстрый старт

```bash
npm install
npm start
```

Затем откройте **http://localhost:4173** в браузере.
(Сменить порт: `PORT=5000 npm start`.)

## 🧩 Что умеет

- 🗂️ **Умная категоризация** — изображения, видео, музыка, документы, код, **модели** (веса ML), **базы данных**, архивы, **ярлыки**, **конфиги** и другое — плюс любая категория, созданная вами или ИИ.
- 🛡️ **Сохраняет целостность** проектов, приложений, баз данных и сохранённых веб-страниц.
- 🤖 **Локальная LLM (опционально)** — классификация неизвестных типов и **финальная проверка всех файлов**.
- 🔁 **Поиск дубликатов** — точные (по хешу содержимого) и визуально похожие фото.
- 🧹 **Очистка** `node_modules`, кэшей сборки и системного мусора.
- 📸 **Умные подпапки** — фото по дате EXIF, музыка по исполнителю/альбому.
- 👀 **Проверка перед перемещением** — весь план, редактирование, исключения.
- ⏪ **Откат + снимки** структуры папок для каждого запуска.
- 💾 **Автоматическое возобновление** после сбоя.
- 📊 **Живой прогресс и журнал в реальном времени.**

## 🔒 Приватность

Это персональная настольная утилита: без входа, без многопользовательского режима, без облака, без телеметрии. Все данные (выученные правила категорий, снимки сканирования, история запусков для отмены) хранятся в локальной папке `data/` рядом с приложением. Если включить опциональную LLM, запросы идут только на тот локальный эндпоинт, который настроите **вы**.
