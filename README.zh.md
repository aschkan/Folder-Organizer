# 📁 Folder Organizer(文件夹整理器）

**一款本地、私密、AI 辅助的工具，可安全地将最混乱的文件夹按类别整理——绝不破坏你的应用程序、代码项目或数据库——并支持一键完整回滚。**

一切都在你自己的电脑上运行。不上传任何内容。在你审阅并确认之前，不会移动任何文件。

## ✨ 与众不同之处

大多数“文件整理工具”只按扩展名把所有东西铺平到文件夹里，这样会**破坏**那些只能作为整体运行的东西：一个代码项目、一个便携应用、一个数据库、一个已保存的网页。本应用理解结构：

- 🛡️ **保持整体单元完整。** 代码项目（`package.json`、`.git`……）、应用程序（便携应用、浏览器、模拟器、游戏——甚至没有扩展名的 Linux/macOS 可执行文件）、数据库（SQLite、LevelDB/RocksDB）以及已保存的网页（`..._files` 文件夹）都会作为**一个整体**移动，绝不打散。
- 🤖 **由 AI 驱动，而非固定列表。** 类别并非硬编码。可选的**本地 LLM** 会结合完整路径上下文对未知文件类型进行分类，并能创建新类别——而且在每次扫描结束时，它会**重新审阅整个方案以纠正自己的错误**。
- ⏪ **完全可逆。** 每次运行都会生成快照；一键即可把一切精确还原到原处。
- 💾 **防崩溃。** 如果电脑或应用在中途关闭，重启后会从中断处精确继续。
- 🔒 **100% 本地且私密。** 无云端、无遥测、无账号。

## 🚀 快速开始

```bash
npm install
npm start
```

然后在浏览器中打开 **http://localhost:4173**。
（用 `PORT=5000 npm start` 更改端口。）

## 🧩 功能

- 🗂️ **智能分类** — 图片、视频、音乐、文档、代码、**模型**（机器学习权重）、**数据库**、压缩包、**快捷方式**、**配置**等——以及你或 AI 创建的任何类别。
- 🛡️ **保持完整**：项目、应用程序、数据库和已保存的网页。
- 🤖 **本地 LLM（可选）** — 对未知类型分类，并对**所有文件进行最终复核**。
- 🔁 **重复检测** — 精确重复（按内容哈希）与视觉相似的照片。
- 🧹 **清理** `node_modules`、构建缓存和系统垃圾文件。
- 📸 **智能子文件夹** — 照片按 EXIF 日期，音乐按艺人/专辑。
- 👀 **移动前先审阅** — 查看完整方案、编辑、排除任意文件。
- ⏪ **回滚 + 快照**：为每次运行保存文件夹结构快照。
- 💾 **中断后自动续跑。**
- 📊 **实时进度与活动日志。**

## 🔒 隐私

这是一款个人桌面工具：无登录、无多用户、无云端、无遥测。所有状态（已学习的分类规则、扫描快照、用于撤销的运行历史）都保存在应用旁边的本地 `data/` 文件夹中。如果启用可选的 LLM，请求只会发送到**你**配置的本地端点。
